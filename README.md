# 📱 Daily Company

Aplicativo mobile desenvolvido para auxiliar empresas na organização de tarefas e atividades do dia a dia.

O **Daily Company** permite que equipes registrem, acompanhem e gerenciem tarefas corporativas de forma simples e organizada, facilitando o controle das responsabilidades dentro da empresa.

---

# 📌 Sobre o projeto

O **Daily Company** é um sistema mobile voltado para a gestão de tarefas empresariais.

A proposta do aplicativo é ajudar equipes a manterem suas atividades organizadas, permitindo o acompanhamento das tarefas e melhorando a produtividade no ambiente de trabalho.

Este projeto também tem como objetivo praticar desenvolvimento de aplicações mobile e organização de código utilizando Git e GitHub.

---

# 🚀 Funcionalidades

* 📋 Cadastro de tarefas
* 📊 Visualização das tarefas registradas
* ✅ Controle de tarefas concluídas
* 🗂 Organização das atividades da empresa
* 📅 Acompanhamento das tarefas do dia a dia

# ⚙️ Como executar o projeto

### 1️⃣ Clone o repositório

```
git clone https://github.com/miguel-oliveira-git/Daily_Company.git
```

### 2️⃣ Acesse a pasta do projeto

```
cd Daily_Company
```

### 3️⃣ Instale as dependências

```
flutter pub get
```

### 4️⃣ Execute o projeto

```
flutter run
```
